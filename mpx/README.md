# MPX 核心代码模块

本目录是整个项目的**代码库中枢根目录 (Source Root)**。在这里下挂了控制循环所需的算法求解器与仿真执行流程的具体Python模块实现。整个目录支持作为独立 PIP 模块安装。

## 该层级下的下级组织：
- **`primal_dual_ilqr/`**: 底层基于 JAX GPU 并行的高速最优求解算法（序列二次优化/时变 LQR核心）。
- **`utils/`**: 将机器人模型、地形、代价函数以及底层优化器粘合在一起的控制器外皮和数学处理子功能。
- **`examples/`**: 单环境及批处理环境中，对于上述各个控制器使用演示的方法汇集。
- **`config/`**: 各类独立机器人对象以及环境的任务硬编码超参数配置表及目标惩罚权重集合。
- **`data/`**: 支撑运行的机器人 3D 机械树本体资源及环境信息包。
