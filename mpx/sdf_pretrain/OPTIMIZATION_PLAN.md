## 2. 问题根因分析

### 2.1 致命问题：采样分布错误

**当前策略**: 只在地形外部采样（SDF > 0）

**问题**:
1. **几何界定崩溃**: 网络从未见过表面(SDF=0)和内部(SDF<0)，无法学习边界位置
2. **MPC灾难**: 轻微穿模时(SDF<0)，网络输出不可预测，导致机器人失控

```
当前采样分布:
┌─────────────────────────────────────────────┐
│  ████████████████████████████████  外部     │ ← 100% 采样点
│  ░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░  表面±5cm │ ← 0% 采样点！
│  ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒  内部     │ ← 0% 采样点！
└─────────────────────────────────────────────┘
```

### 2.2 次要问题

| 问题 | 影响 | 优先级 |
|------|------|--------|
| MLP难以学习高频特征 | 锐利边缘变圆滑 | 高 |
| Eikonal Loss频率过高 | 可能干扰MSE下降 | 中 |
| 网络容量可能不足 | 无法表达复杂地形 | 低 |
| 采样点数量 | 覆盖不够密集 | 低 |

## 3. 解决方案

### 3.1 核心修复（优先级最高）

#### A. 表面重要性采样

```python
# 新的采样分布
采样比例:
  - 10% 内部:  SDF ∈ [-0.10, 0)       → 穿模检测
  - 70% 表面:  SDF ∈ [-0.05, +0.05]   → 核心区域！机器人脚部活动区
  - 20% 外部:  SDF ∈ [+0.05, +0.50]   → 全局引导

实现方式:
  1. 先计算所有采样点的SDF
  2. 根据SDF值分类到三个区间
  3. 按比例保留/补充采样点
```

#### B. 位置编码 (Positional Encoding)

```python
# NeRF风格的傅里叶特征
def positional_encoding(p, L=6):
    """
    输入: p = (x, y, z), shape (..., 3)
    输出: γ(p), shape (..., 3 * 2 * L)
    """
    encoded = []
    for i in range(L):
        freq = 2 ** i
        encoded.append(jnp.sin(freq * jnp.pi * p))
        encoded.append(jnp.cos(freq * jnp.pi * p))
    return jnp.concatenate(encoded, axis=-1)

# 例如 L=6: 输入3维 → 输出36维
# 让MLP能够学习锐利的高频特征（台阶直角）
```

### 3.2 次要调整

#### C. Eikonal Loss策略

```bash
# 当前配置
EIKONAL_WEIGHT=0.1
EIKONAL_EVERY=1

# 建议配置（减少对MSE的干扰）
EIKONAL_WEIGHT=0.05
EIKONAL_EVERY=5
```

#### D. 网络与采样（如果A+B+C不够）

```bash
# 可选调整
LATENT_DIM=512      # 256 → 512
NUM_QUERIES=2048    # 1024 → 2048
```
