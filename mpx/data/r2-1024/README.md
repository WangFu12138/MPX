# R2-1024 数据目录说明

本目录主要是一个机器人的**导出包与模型映射结构资产文件 (ROS-like Structure)**。该配置专门描述 Unitree R2 机器人（或者是类似比例版本模型）的几何尺寸与动力学限位信息。

## 基础内容体系：
- **`mjcf/` & `urdf/`**: 存放了包含机械连杆 (Links)、关节结构 (Joints)、碰撞框结构矩阵的核心 XML 模型主干及其切分的依赖。
- **`config/` 和其它生成物如 `export.log`, `package.xml`, `CMakeLists.txt`**: 表明该物理资产很可能是从现有 URDF 资产树或其他机器人标准代码库转换/移植并导出兼容进入本工程引擎格式的产物。
